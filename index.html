The Problem

The same home page. For everyone.

Most investing apps show every user the same home page — same widgets, same asset-class tiles, same tools — whether you're a first-time SIP investor or an active F&O trader.

Relevant products stay buried. Communications feel like spam.

The MO Personalization Suite closes that gap — one behavioural foundation that reorders every widget per user and tailors communications across push, email, SMS, and WhatsApp. Live on the RIISE app today.

Architecture

One foundation → two surfaces

A single feature store and an 18-segment behavioural taxonomy power both home-page personalization and omnichannel communications. A ranker scores every widget; a re-ranker adds diversity, popularity, and cross-sell.

How It Works

A behavioural foundation and a two-stage ranker

Two components do the work: a behavioural foundation that turns hundreds of features into clear segments, and a two-stage ranker that surfaces the widgets each user actually wants.

Live Deployment

Live on the RIISE app

The same feature store and 18-segment taxonomy drive per-user widget order on the home page and tailored content across four channels — push, email, SMS, and WhatsApp. Next, we're extending it to every RIISE surface.
